S3 ninja

S3 ninja emulates the S3 API for development and testing purposes.

Download Contribute
Screenshot
Use
  1. A readily packaged docker image is available at scireum/s3-ninja
  2. Run like docker run -p 9444:9000 scireum/s3-ninja:6
  3. Navigate to http://localhost:9444/ui
  4. Run S3 API-Calls against http://localhost:9444/ (e.g. http://localhost:9444/test-bucket/test-object)
  5. Provide an volume for /home/sirius/data to persist data accross restarts.
API
Currently the ninja supports only object methods: GET, PUT, HEAD, DELETE. ACLs are also ignored but for non-public buckets, the given Authorization hash is verified using the credentials shown on the page.
